A) HTML Structure - top

This one page site template is a fixed-width design (890px) Use it for your business, your portfolio, some products or any other purpose.

How to change the logo:
Replace the mainlogo by replacing the file with the name: logo.png. You can find the logo.psd file in the psd folder.

How to change the images in the nivo slider:
To add more images to Nivo slider simply find the comment < !-- start nivo slider -- > and then insert your portfolio images. At this moment there are 3 placeholders called: slider01.jpg, slider02.jpg and slider03.jpg. Of course, you can replace these images. To put a simple caption use the tag title; if you wont an advanced caption, modify (or use multiple id name of this) find the comment < !-- start nivo slider caption-- > .

How to put your own navigation:
Search for the psd-file: "nav.psd" - You only have to write your own words with every font you want. (I used BEBAS NEUE FONT) Then save your navi as nav.png and overwrite the original file in the images folder.

How to put your own video:
Search for <!-- Your video here --> in the main code and replace the link with your own.

How to put your own portfolio images:
Search for <!-- Your portfolio images start here --> and replace the images (1.jpg, 1_black.jpg, ...) in the portfolio-folder with your own images.

How to change the text area:
Search for <!-- Your text areas starts here --> and write every text you want.

How to change the cufon-font:
You have to go http://cufon.shoqolate.com/generate/ and upload every font you want to use. Then save your file in the js folder. Search for <!--START CUFON--> in the main code and replace Droid_Sans_400.font with your generated file name. Done!

How to enable the tooltip:
To place a tooltip above your images or text you only have to use this code:
<a href="#" class="tip_trigger">YOUR IMAGE OR TEXT <span class="tip">This will show up in the tooltip</span></a></div>

How to change the list styles:
Search for <!-- Lists--> in the maincode and write your lists.

How to change the tabbed content:
Search for <!--Tabbed Boxes--> and change the text. To add a new box you only have to write copy content_3 and rename it in content_4. Than add the content_4 in the css menu and the ul.
